RDA value vocabularies are recommended for use as Vocabulary Encoding Schemes for specified RDA properties.

The vocabularies contain terms, definitions, and scope notes in English.

The vocabularies contain translations of terms, definitions, and scope notes in:

  • French
  • German
  • Spanish

Translations may be partial or out-of-date.

The JSC will publish new, agreed value vocabularies in the rdaregistry.info namespace.
